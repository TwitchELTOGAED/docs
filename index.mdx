# La Cochera de la Nieve - Documentación Técnica

Bienvenidos al manual de arquitectura y mantenimiento del sistema multiplataforma desarrollado para **La Cochera de la Nieve**. Este proyecto supone la transformación digital de un comercio local, ofreciendo una solución moderna, escalable y centralizada.

## Arquitectura del Sistema

El proyecto sigue una arquitectura cliente-servidor dividida en tres pilares fundamentales que operan de forma sincronizada:

* **Frontend Web (PWA):** Desarrollado con **Next.js (React)** y desplegado en la nube mediante **Vercel** con Integración y Despliegue Continuo (CI/CD). Garantiza un SEO óptimo y tiempos de carga mínimos.
* **Frontend Móvil:** Aplicación nativa para Android desarrollada en **Kotlin** utilizando el moderno framework declarativo **Jetpack Compose**.
* **Backend as a Service (BaaS):** Centralizado en **Supabase** (PostgreSQL). Se encarga de gestionar la autenticación de usuarios, la base de datos relacional y el almacenamiento (Storage) de las imágenes del catálogo en la nube.
* **Procesamiento de Pagos:** Integración segura con la pasarela **Stripe** para procesar transacciones garantizando el cumplimiento de la normativa PCI-DSS, delegando el manejo de datos sensibles.

## Funcionalidades Core

* **Sincronización Multiplataforma:** Cualquier modificación del inventario en la base de datos se refleja instantáneamente tanto en la plataforma web como en la aplicación móvil.
* **Gestión Integral de Pedidos (Back-office):** Panel de administración exclusivo para gestionar el ciclo de vida de las compras (Pendiente, Enviado, Entregado) con sistema de notificaciones integrado.
* **Catálogo Dinámico (CRUD avanzado):** Creación, edición en caliente y sistema de borrado lógico (ocultación) de productos para gestionar eficientemente los cambios de temporada o falta de stock temporal.
